# 📝 Changelog

Histórico de versões e mudanças do ITWOMA.

## [1.0.0] - 2026-05-20

### 🎉 Inicial Release

#### Adicionado
- ✨ Cliente CLI completo para MangaDex
- ✨ Busca de mangás em tempo real
- ✨ Download automático de capítulos
- ✨ Suporte para português e inglês
- ✨ Visualizador integrado
- ✨ Limpeza automática de arquivos
- ✨ Suporte multiplataforma (Linux, macOS, Windows)
- ✨ Build com CMake
- ✨ Parser JSON com cJSON
- ✨ Cliente HTTP com libcurl

#### Documentação
- 📖 README.md completo
- 📖 Guia rápido (QUICKSTART.md)
- 📖 Guia de desenvolvimento (DEVELOPMENT.md)
- 📖 Guia de contribuição (CONTRIBUTING.md)
- 📖 Análise de projeto (PROJECT_ANALYSIS.md)
- 📖 FAQ (FAQ.md)
- 📖 Este Changelog

#### Infraestrutura
- 🔧 CMakeLists.txt otimizado
- 🔧 .gitignore apropriado
- 🔧 .editorconfig para consistência
- 🔧 Suporte para Debug builds
